## 🚀 Как Запустить

### Шаг 1: Запустить Docker Desktop
Убедитесь, что Docker Desktop запущен и работает.

### Шаг 2: Пересоздать БД (рекомендуется)

```bash
# Остановить контейнеры
docker-compose down

# Удалить старую БД
docker volume rm ivrit_postgres_data

# Запустить заново
docker-compose up -d
```

Backend автоматически:
1. Применит миграции
2. Запустит основной seed (10 уроков, 50+ слов, 18+ фраз)

### Шаг 3: Добавить Расширенный Контент (опционально)

```bash
# Зайти в контейнер backend
docker exec -it ulpan-backend sh

# Запустить расширенный seed
npm run prisma:seed:extended

# Выйти
exit
```

Это добавит ещё 200+ слов и 30+ фраз.

### Альтернатива: Локальный запуск

```bash
cd backend

# Основной seed
npm run prisma:seed

# Расширенный seed
npm run prisma:seed:extended
```
