## Как Запустить

### Вариант 1: Пересоздать БД с нуля (рекомендуется)

```bash
# Остановить контейнеры
docker-compose down

# Удалить volume с БД
docker volume rm ivrit_postgres_data

# Запустить заново
docker-compose up -d

# Дождаться запуска backend (он автоматически запустит seed)
```

### Вариант 2: Добавить только новые данные

```bash
# Зайти в контейнер backend
docker exec -it ulpan-backend sh

# Запустить расширенный seed
npm run prisma:seed:extended

# Выйти
exit
```

### Вариант 3: Локальный запуск расширенного seed

```bash
cd backend
npm run prisma:seed:extended
```
