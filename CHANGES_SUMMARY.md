## 🚀 Как Применить Изменения

### Вариант 1: Полная Пересборка (Рекомендуется)

```bash
# Остановить контейнеры
docker-compose down

# Удалить старую БД
docker volume rm ivrit_postgres_data

# Запустить заново
docker-compose up -d
```

### Вариант 2: Только Расширенный Контент

```bash
# Зайти в backend
docker exec -it ulpan-backend sh

# Запустить расширенный seed
npm run prisma:seed:extended

# Выйти
exit
```
