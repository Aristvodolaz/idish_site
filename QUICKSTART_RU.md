# Быстрый Старт - Ulpan AI (Русская Версия)

## 🚀 Запуск за 3 Шага

### 1. Запустите Docker Desktop
Убедитесь, что Docker Desktop запущен.

### 2. Пересоздайте БД с новым контентом

```bash
docker-compose down
docker volume rm ivrit_postgres_data
docker-compose up -d
```

### 3. Откройте приложение

Подождите 30-60 секунд, затем откройте:
- **Frontend:** http://localhost:3000
- **Backend API:** http://localhost:4000

## 🎯 Как Использовать

1. **Зарегистрируйтесь** на http://localhost:3000/register
2. **Войдите** в аккаунт
3. **Выберите "Учить Слова"** (зелёная кнопка на Dashboard)
4. **Начните учить!** Переворачивайте карточки, нажимайте "Знаю"/"Не знаю"

## 🔧 Добавить Ещё Больше Слов (Опционально)

Если хотите добавить ещё 200+ слов:

```bash
docker exec -it ulpan-backend sh
npm run prisma:seed:extended
exit
```
